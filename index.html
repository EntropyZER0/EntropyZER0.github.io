Terminologia

HTML

Lenguaje de marcado utilizado para estructurar y presentar contenido en la web. HTML significa "HyperText Markup Language".

CSS

Lenguaje de estilos utilizado para describir la apariencia y el diseño de una página web. CSS significa "Cascading Style Sheets".

JavaScript

Lenguaje de programación utilizado para añadir interactividad a las páginas web, como animaciones y validaciones de formularios.

API

Interfaz de programación de aplicaciones que permite la comunicación entre diferentes programas o servicios. API significa "Application Programming Interface".

URL

Localizador uniforme de recursos utilizado para especificar la ubicación de un recurso en Internet. URL significa "Uniform Resource Locator".

WWW

World Wide Web, es el sistema de documentos interconectados y accesibles en Internet a través de enlaces y URLs.
DOM
Modelo de objetos del documento, una representación estructurada de un documento HTML o XML en el navegador.

Front-End

Parte visible y con la que interactúa el usuario en una aplicación o sitio web. Incluye HTML, CSS y JavaScript.

Back-End

Parte de un sistema que procesa la lógica de la aplicación y gestiona la base de datos. No es visible para el usuario.

Framework

Conjunto de herramientas y bibliotecas que proporcionan una estructura base para el desarrollo de software.

Bootstrap

Framework de CSS y JavaScript para el desarrollo de sitios web responsivos y móviles.

Git

Sistema de control de versiones utilizado para gestionar y realizar seguimiento a los cambios en el código fuente.

GitHub

Plataforma basada en la web para alojar repositorios Git, facilitando la colaboración y el control de versiones.

Repositorio

Directorio donde se almacenan los archivos de un proyecto y su historial de cambios.

IDE

Entorno de desarrollo integrado que proporciona herramientas para escribir, compilar y depurar código. IDE significa "Integrated Development Environment".

Compilador

Programa que traduce el código fuente de un lenguaje de programación a lenguaje máquina.

Algoritmo

Serie de instrucciones o pasos para resolver un problema específico o realizar una tarea.

Base de Datos

Sistema organizado para almacenar, gestionar y acceder a datos de manera eficiente.

SQL

Lenguaje de consulta estructurado utilizado para gestionar y manipular bases de datos. SQL significa "Structured Query Language".

NoSQL

Tipo de base de datos que no utiliza el modelo de tablas relacionales y se enfoca en el almacenamiento de datos no estructurados.

JSON

Formato de datos ligero utilizado para intercambiar información entre servidores y aplicaciones web. JSON significa "JavaScript Object Notation".

REST

Estilo de arquitectura para la creación de servicios web, basado en operaciones HTTP como GET, POST, PUT y DELETE.

CRUD

Acrónimo de Crear, Leer, Actualizar y Eliminar. Representa las operaciones básicas en bases de datos y aplicaciones web.

Script

Archivo de texto que contiene código de programación ejecutable, generalmente utilizado para automatizar tareas.

IP

Dirección de Protocolo de Internet utilizada para identificar dispositivos en una red. IP significa "Internet Protocol".

DNS

Sistema de nombres de dominio que traduce nombres de dominio en direcciones IP. DNS significa "Domain Name System".

VPN

Red privada virtual que permite conexiones seguras y privadas a través de redes públicas. VPN significa "Virtual Private Network".

Bug

Error o defecto en un programa de software que causa resultados inesperados o problemas.

Debugging

Proceso de identificar y corregir errores en el código de un programa.

Machine Learning

Rama de la inteligencia artificial que permite a los sistemas aprender y mejorar a partir de la experiencia.
Cloud Computing
Modelo de computación que permite el acceso a recursos y servicios a través de Internet en lugar de en hardware local.

SDK

Kit de desarrollo de software que proporciona herramientas y bibliotecas para desarrollar aplicaciones en una plataforma específica.

Cookie

Pequeño archivo de datos almacenado en el navegador web del usuario que guarda preferencias y sesiones de usuario.

Cache

Memoria temporal que almacena copias de datos o archivos para mejorar la velocidad de acceso en futuras solicitudes.

UX

Experiencia de usuario, se refiere a cómo se siente el usuario al interactuar con un sistema o aplicación.

UI

Interfaz de usuario, es el diseño visual y la disposición de elementos con los que interactúa el usuario.

Binary

Sistema numérico que utiliza solo los dígitos 0 y 1, base fundamental para la computación.

Object-Oriented Programming (OOP)

Paradigma de programación que organiza el código en objetos que representan tanto datos como funcionalidades.

Array

Estructura de datos que almacena una colección de elementos del mismo tipo.

Variable

Espacio en memoria donde se almacena un valor que puede cambiar durante la ejecución de un programa.
